# Chrome Web Store Listing Copy

## Name

`Leave Me Alone`

## Short description

Hide Facebook Reels, Stories, video surfaces, and YouTube Shorts so your feed stays quieter.

## Full description

Leave Me Alone helps reduce common doomscroll traps on Facebook and YouTube.

Use simple switches to hide:

- Facebook Reels
- Facebook video surfaces
- Facebook Stories
- YouTube Shorts

The extension is intentionally narrow in scope. It only runs on Facebook and YouTube, and it only hides the supported feed surfaces listed above.

Your settings are saved locally in the browser so the extension remembers what you want hidden.

## Single purpose statement

Leave Me Alone hides selected doomscroll-oriented feed surfaces on Facebook and YouTube.

## Permissions justification

- `storage`: saves the user's chosen hide/show settings locally in the browser.
- Facebook and YouTube host access: allows the extension to identify and hide the supported surfaces on those pages only.
