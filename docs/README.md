# Leave Me Alone Docs

Leave Me Alone reduces common doomscroll bait on Facebook and YouTube by hiding selected surfaces:

- Facebook Reels
- Facebook video surfaces
- Facebook Stories
- YouTube Shorts

## Public docs

- [Privacy Policy](./privacy-policy.md)
- [Support](./support.md)

## What the extension stores

The extension stores your toggle settings locally in the browser so it remembers which supported surfaces you want hidden.
