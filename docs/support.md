# Leave Me Alone Support

## Need help?

If a supported surface is not hiding correctly, capture the page URL, which toggles were enabled, and a screenshot of what you expected to happen.

## Supported behavior

The extension currently targets Facebook Reels, Facebook video surfaces, Facebook Stories, and YouTube Shorts on the domains declared in the extension manifest.

## Reporting an issue

Open an issue in the project repository or use the publisher support contact listed on the Chrome Web Store item.

## Reviewer note

Some Facebook-specific surfaces depend on the current logged-in Facebook experience and may vary by account, region, or experiment bucket.
